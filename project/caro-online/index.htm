Description:

Game Caro Vietnam

A casual puzzle game based on the classic five-in-a-row concept, redesigned for a smooth and accessible player experience.

My Role:
I joined the project from its early stages, where I was responsible for both UI system design and core gameplay implementation. I worked on building an intuitive user interface, managing game states, and ensuring responsive, bug-free turn-based logic.

What I Do:
  • Designed and implemented the full UI system.
  • Developed the core gameplay logic including turn-based mechanics, move handling, and win/loss detection.
  • Managed game state transitions such as waiting for turn, playing, and game over.
What I Learn:

  • Gained a clear understanding of how turn-based games work under the hood.

  • Learned how to manage turn logic and maintain smooth gameplay flow.

  • Improved skills in building intuitive UI for casual puzzle games.

  • Understood how to take a simple game concept from initial idea to a complete, playable product.
